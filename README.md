# Customizing the Portfolio Template

Welcome to the portfolio template! Follow these instructions to customize it according to your preferences.

## 1. Update Your Information
- Open the `data.js` file and replace the placeholder information with your personal details.
- Make sure to include your name, bio, and links to your social media accounts.

## 2. Change Color Scheme
- Navigate to `styles.css` to modify the color variables to match your brand.
- Adjust the colors for the background, text, and buttons as needed.

## 3. Add Your Projects
- Add details about your projects in the `projects` array found in `data.js`.
- Include project title, description, and links to live demos or source code.

## 4. Update the Layout
- If you want to change the layout, feel free to modify the HTML structure in the `index.html` file.
- Be cautious when changing class names used in CSS and JS.

## 5. Test Your Changes
- Open the `index.html` file in your browser to preview your customized portfolio.
- Make sure everything looks good and functions correctly!

Happy customizing!
